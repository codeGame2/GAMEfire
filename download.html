Some steps to install for Windows


  1. Install the .zip file.
  2. Extract the downloaded file.
  3. Move extracted folder to "program Files" in the path:
    C:/program Files
  4. Open the folder.
  5. Right-click on "GAMEfire.exe" and select copy.
  6. Then, goto Desktop.
  7. Press shift + ctrl + v to paste the shortcut.
    NOTE: It is not please don't press ctrl + v or paste
